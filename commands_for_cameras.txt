cd to ros2_ws

for each node open a new terminal and cd to ros2_ws

to initialize camera_front node:
ros2 launch zed_wrapper zed_camera_front.launch.py camera_model:=zed2i

to initialize camera_rear node:
ros2 launch zed_wrapper zed_camera_rear.launch.py camera_model:=zed2i

ros2 run zed2i_data_collector

ros2 run zed2i_data_collector data_collector --ros-args -p dataset_location:=~/tau_dataset -p camera_direction:=front

ros2 run zed2i_data_collector data_collector --ros-args -p dataset_location:=~/tau_dataset -p camera_direction:=rear

to see current topics list:

ros2 topic list
